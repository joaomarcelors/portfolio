Projeto para um Sistema de Manufatura

Descrição

Este projeto foi desenvolvido em equipe com meus amigos Vitor E. F. Oliveira e Elianfrancis S. Souza, na disciplina de Sistemas Integrados de Manufatura, durante minha graduação. A atividade foi proposta pelo professor Rodolfo L. Valle e tinha como objetivo programar toda a planta CIM-C, simulando uma automação de um processo industrial completo. O sistema deveria receber uma peça polímero ou uma metálica de aluminio ou de aço, que passaria por diversas etapas, como furação, inspeção do furo por visão computacional e pesagem. Também foram implementadas rotinas para os botões de START, STOP e EMERGÊNCIA, além do tratamento de situações adversas, como reprovação da peça durante o processo. Ao final, foi desenvolvido um supervisório para monitorar o sistema em tempo real.

Inicialmente, foi estudado todos os sensores e atuadores conectados ao CLP (modelo Twido – Schneider Electric). Em seguida, realizamos o mapeamento das memórias de entradas e saídas em uma tabela para organização e referência. Para estruturação da lógica de programação, foi utilizado o linguagem Grafcet, que posteriormente foi convertido para a linguagem Ladder e implementado no software TwidoSuite. O supervisório foi desenvolvido no software Eclipse E3, utilizando o protocolo de comunicação MODBUS. Ele apresentava informações como total de peças produzidas, número de erros, peças rejeitadas e mensagens de alerta para o operador, garantindo melhor acompanhamento do processo.

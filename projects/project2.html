Automação para irrigação

Descrição

Esse projeto teve como objetivo automatizar o todo o processo de irrigação, possuindo diversas funcionalidades.O sistema foi implementado utilizando um microcontrolador Arduino MEGA programado no Arduino IDE, a escolha desse Arduino foi devido a quantidade de portas e armazenamento de memória.Possui capacidade de configurar até 4 programas diferentes que podem ser escolhido para ser executado. O sistema conta com um módulo relógio para obter data e hora atuais, um display LCD 20x4 para visualização das informações, botões para interagir com o sistema, relés para acionamentos da bomba, setores, fertilizante e limpeza.As opções de seleção do sistema são:

  1. Sistema desligado.
  2. Ajuste de data e hora do sistema.
  3. Horários para iniciar a irrigação para cada programa.
  4. Tempo da irrigação para cada um dos 16 setores e também para cada programa.
  5. Inicio de injeção de fertilizante no setor, bem como o volume e o tempo máximo de operação da bomba do fertilizante.
  6. Quais dias da semana, o sistema vai realizar a irrigação para cada programa.
  7. Tempo para iniciar a limpeza e quanto tempo de limpeza.
  8. Inicia a irrigação para um setor especifico.
  9. Executa um programa de imediato.
  10. Executa um programa dado o horário e o dia configurado.

O sistema se mostrou eficaz e confiável, considerando diversas situações adversas que podem vim a comprometer o funcionamento pleno do sistema.

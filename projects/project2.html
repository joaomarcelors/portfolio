Automação para irrigação

Descrição

Esse projeto teve como objetivo automatizar o todo o processo de irrigação como, possuindo diversas funcionalidades.O sistema foi implementado utilizando um microcontrolador Arduino MEGA programado no Arduino IDE, a escolha desse Arduino foi devido a quantidade de portas e capacidade de memória.O sistema possui um módulo relógio para obter data e hora atuais, um display LCD 20x4 para visualização das informações, botões para interagir com o sistema, relés para acionamentos da bomba, setores, fertilizante e limpeza.As opções de seleção do sistema são:

  1. O sistema interrompe,se necessário, quaisquer tarefas que esteja acontecendo no momento.
  2. Configuração de data e hora do sistema.
  3. Configuração do horário para iniciar a irrigação para cada programa.
  4. Configuração da duração da irrigação para cada um dos 16 setores e também para cada programa.
  6. A partir de um cartão SD foi possível salvar as informações em momentos que não havia conexões com a internet, sendo enviadas para a nuvem posteriormente.
  7. O sistema se mostrou eficaz, com alta velocidade de transmissão de dados e sem perdas de informações.

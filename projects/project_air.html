Controle inteligente para ar condicionado

Descrição

Este é um sistema inteligente com o objetivo controlar o ar condicionado das salas do CEFET-MG Campus Leopoldina, sem a necessidade do controle remoto original.Esse projeto foi desenvolvido em equipe na disciplina de Projeto do Produto/Processo durante minha graduação. Fiquei responsável pela programação e auxilio na montagem do protótipo.

  • O sistema foi implementado utilizando um microcontrolador ESP32 programado no Arduino IDE, foi usado o protocolo HTTP (Hypertext Transfer Protocol) para comunicação entre o dispositivo e o computador a partir da conexão Wi-Fi.
  • O sistema possui um modulo RFID, para que apenas professores ou alunos autorizados que tenham cartão cadastrado possam usar o dispositivo.O cadastrado de novos usuários era feito em uma planilha no computador e por meio de um script em python eram enviados as tags dos cartões dos usuários para o ESP32. Todos as tags eram salvos na memória interna do ESP32.
  • A partir de um led infra-vermelho era possível captar os comandos de liga/desliga e temperaturas do controle original de cada ar condicionado.Assim, com esses comandos eram possível controlar o ar condicionado pelo dispositivo
  • Ao aproximar o cartão RFID o sistema liberava ou não o acesso ao usuário.Todas as informações eram mostradas em display LCD 16x2, o dispositivo possui três botões para ligar/desligar, diminuir e aumentar a temperatura do ar.Além de possuir um led verde e um buzzer para interatividade com o usuário.
  • O usuário ao ligar o ar-condicionado um timer de 2h era acionado, passado esse tempo,o sistema automaticamente enviava um sinal para desligar o ar. Esse tempo estipulado foi com base no tempo de uma aula normal.
  • Durante os testes, o sistema demonstrou-se funcional e está em andamento para a efetiva instalação nas salas.
